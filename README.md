# Krishna Fast Food & Biryani Point 🍛

A **responsive web application** for Krishna Fast Food & Biryani Point — the best biryani in Medchal! Built with pure HTML, CSS and JavaScript.

## ✨ Features

- 🍽️ **Full menu display** with food items
- ⭐ **Customer reviews** section
- 📞 **Contact section** for orders
- 📱 **Responsive design** — works on all devices
- 🛒 **Order button** integration
- 🧭 **Navigation bar** — Home, Menu, Reviews, Contact

## 🚀 How to Run

```bash
git clone https://github.com/Eshwarkadari/KRISHNA_FAST_FOOD_CENTER.git
cd KRISHNA_FAST_FOOD_CENTER
```

Simply open `index.html` in your browser — no server needed!
